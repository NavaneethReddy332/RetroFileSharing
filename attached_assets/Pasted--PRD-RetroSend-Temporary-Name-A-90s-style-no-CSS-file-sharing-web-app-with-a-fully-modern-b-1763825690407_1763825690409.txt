1. PRODUCT OVERVIEW

RetroSend is a simple, fast, temporary file-sharing service inspired by SendAnywhere, but with a pure old-school 90s GUI (HTML only, inline fonts, tables, no CSS framework).
Backend will be modern, secure, API-driven, and support file uploads, unique share codes, and download links.
